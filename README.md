# 거위 시뮬레이터
> 몰입캠프 4주차 프로젝트

* 홈 파티를 위한 티켓을 생성해주는 Android 기반 어플리케이션입니다.  
* 홈 파티 멤버들의 연락처와 주소를 저장할 수 있습니다.  
* 티켓을 만들고, 바코드를 통해 멤버들과 공유가 가능합니다.  
* 만들어진 홈 파티 티켓을 저장하고 관리할 수 있습니다.
